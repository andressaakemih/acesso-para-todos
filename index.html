Acessibilidade web

Para a maioria das pessoas, a tecnologia torna a vida mais fácil. Para uma pessoa com necessidades especiais, a tecnologia torna as coisas possíveis

Jorge Fernandes e Francisco Godinho (2003)

Sobre acessibilidade web

Existem muitas definições de acessibilidade web, uma delas é possibilitar às pessoas com deficiência tenham autonomia com acesso a uma web projetada de modo que possam navegar, ter acesso a conteúdos, interagir e contribuir.

A W3C define uma web acessível como:

"Acessibilidade na Web significa que sites, ferramentas e tecnologias são projetados e desenvolvidos para que pessoas com deficiências possam usá-los. Mais especificamente, as pessoas podem: a) perceber, entender, navegar e interagir com a Web; b) contribuir para a web."
